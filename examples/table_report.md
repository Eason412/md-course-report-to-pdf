# 表格排版回归示例

## 摘要

本文用一个最小表格检查 Markdown pipe table 和紧邻表题在 LaTeX 转换后的编号、表题和三线表格式。

关键词：课程报告；表格；三线表

## Abstract

This minimal report checks whether a Markdown pipe table followed by an adjacent caption is converted into a numbered longtable with booktabs rules.

Keywords: course report; table; booktabs

## 方案比较

课程报告常需要用表格比较多个方案。表格标题、列名和单元格内容应尽量简洁，便于最终 PDF 排版[1]。

| 方案 | 优势 | 约束 |
| --- | --- | --- |
| 方案 A | 易于实现 | 依赖输入质量 |
| 方案 B | 效果稳定 | 成本较高 |
| 方案 C | 扩展性强 | 配置较复杂 |
: 方案对比

## 边界说明

表格只用于格式回归，不代表任何具体项目结论。真实报告应根据课程要求补充数据来源、评价指标和分析过程。
